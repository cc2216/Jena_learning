This project is for implementing query of family tree by rules added.
It allow users to add more rules file for query of family tree ontology.

For using this project, just run it as java application and follow the screen prompt.
The rules files and ontology file should be added in the owl folder.

The reference of the rules files family.rules and family2.rules is http://blog.csdn.net/yangchang999/article/details/6281779
The reference of the family tree ontology is http://www.voidcn.com/article/p-cfoonkdd-bdh.html
